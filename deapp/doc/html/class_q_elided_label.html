Description détaillée

Widget utilitaire permettant d'afficher un texte élagué.

Ce widget permet d'afficher un texte avec un ellipsis (trois points de suspension) si le texte est plus grand que la taille du widget.

Ce widget est présent comme outil simplifiant le développement.

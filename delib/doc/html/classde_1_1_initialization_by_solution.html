Detailed Description

Implémentation de la stratégie de sélection qui conserve les individus ayant la meilleure valeur de fitness.

Cette classe est une stratégie d'initialisation spécifique utilisée dans l'algorithme d'évolution différentielle. Elle initialise les individus par une valeur pseudo-aléatoire répartie uniformément sur le domaine du problème.

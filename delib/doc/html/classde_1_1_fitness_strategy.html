Detailed Description

Classe de base pour les stratégies de fitness.

Cette classe est la classe de base pour les différentes stratégies de fitness utilisées dans l'algorithme d'évolution différentielle. Elle fournit les méthodes de base que chaque stratégie de fitness doit implémenter.

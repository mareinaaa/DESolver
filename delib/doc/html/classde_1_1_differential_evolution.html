Detailed Description

Implémentation de l'algorithme d'évolution différentielle.

L'évolution différentielle est une méthode d'optimisation stochastique qui est utilisée pour optimiser les problèmes d'optimisation à objectif unique et multivariable.

Cette implémentation ne permet pas la copie de l'objet DifferentialEvolution. Si l'utilisateur tente de copier l'objet, une erreur de compilation se produira.

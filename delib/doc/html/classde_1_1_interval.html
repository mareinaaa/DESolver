Detailed Description

Représente le concept mathématique d'un interval dans \(\mathbb{R}\).

Un interval est l'ensemble des nombres disponibles entre deux nombres déterminant les limites de cette plage. On nomme les limites, les bornes. Au sens mathématique, les bornes peuvent être incluses ou excluses de l'interval.

Cette implémentation considère les bornes toujours incluses. Ainsi, on peut dire que le nombre \(x\) satisfait l'interval si \(l \leq x \leq u\) où \(l\) et \(u\) sont les bornes inférieures et supérieures respectivement.

L'interval est valide en tout temps. C'est à dire que les bornes ne peuvent être inversées. Si l'utilisateur tente d'inverser les bornes, une exception sera levée.

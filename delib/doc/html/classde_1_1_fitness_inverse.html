Detailed Description

Stratégie de fitness qui transforme la valeur par inversion.

Cette classe est une implémentation de FitnessStrategy qui effectue une transformation d'inversion 1.0/value sur la valeur donnée. Si la valeur est 0, une valeur de substitution est utilisée à la place.

Detailed Description

La classe de base pour les stratégies dans l'algorithme d'évolution différentielle.

Cette classe fournit une interface commune pour toutes les stratégies utilisées dans l'algorithme d'évolution différentielle. Elle contient des informations de base comme le nom, le résumé et la description de la stratégie.

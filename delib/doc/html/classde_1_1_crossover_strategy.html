Detailed Description

Stratégie générique pour la fonction de croisement dans un algorithme génétique.

Le taux de croisement détermine la probabilité qu'un croisement se produise entre deux individus. Cette classe est destinée à être héritée pour implémenter des méthodes de croisement spécifiques.

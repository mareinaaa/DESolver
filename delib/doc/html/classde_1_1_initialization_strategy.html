Detailed Description

Stratégie d'initialisation pour l'évolution différentielle.

Cette classe est la base pour toutes les stratégies d'initialisation, qui sont utilisées pour assigner la valeur nominal des individus lors du démarrage d'une nouvelle simulation de l'évolution différentielle.
